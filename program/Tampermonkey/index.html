“”用户脚本

    用户脚本是一段代码，它们能够优化您的网页浏览体验。安装之后，有些脚本能为网站添加新的功能，有些能使网站的界面更加易用，有些则能隐藏网站上烦人的部分内容。
    要使用用户脚本，您首先需要安装一个用户脚本管理器。我这里推荐使用“篡改猴”作为您的用户脚本管理器。
    如果您已经安装了“篡改猴”，您可以点击下方“安装此脚本”来安装“”脚本。

安装篡改猴脚本
    电脑端推荐您使用 Microsoft Edge 浏览器 ， 在安装完 Microsoft Edge 后请您用 Microsoft Edge 进入 篡改猴 (Tampermonkey) Edge 外接程序 点击“获取”来安装篡改猴脚本。
    手机端(Android)您可以使用 X浏览器
    手机端(ios)您可以使用Safari
